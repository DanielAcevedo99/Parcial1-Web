# Parcial 1 – TuSegundazo.com 

## Características principales

- **Inicio de sesión** con validación de credenciales.
- **Protección de rutas** mediante guard (solo permite el acceso a `/vehiculos` después del inicio de sesión).
- **Listado de vehículos** obtenido a través de un archivo JSON local.
- **Visualización de detalle** al seleccionar un vehículo.
- **Prueba unitaria** que verifica la correcta creación de la tabla de vehículos.
